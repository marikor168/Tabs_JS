Документация

Функция tabs содержит три аргумента:
tabsTitle - класс табов,
tabsWrapper - класс div, в который "обернуты" табы,
tabsContent - класс div, в которых содержится контент каждого таба.

В вызове функции классы передавать БЕЗ точек, так как в самой функции элементы получаем через getElementsByClassName
